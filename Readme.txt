Descripción breve del sitio: Este proyecto consiste en el desarrollo de un sitio web frontend de una sola página 
(Single Page) utilizando HTML5, CSS3 y Bootstrap 5. El sitio presenta información personal y académica, experiencia,
proyectos realizados, habilidades técnicas, testimonios, contenido multimedia y un formulario de contacto. 
El objetivo es mostrar las competencias adquiridas durante la formación universitaria mediante una interfaz 
moderna, responsiva y profesional.

JavaScript (Bootstrap Bundle)
